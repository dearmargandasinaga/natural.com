Central Processing Unit (CPU)

Central Processing Unit (CPU) adalah otak dari komputer. Ini bertanggung jawab untuk menjalankan instruksi dan mengkoordinasi operasi keseluruhan sistem.

Random Access Memory (RAM)

Random Access Memory (RAM) adalah jenis memori utama dalam komputer yang digunakan untuk menyimpan data dan program secara sementara. Data dalam RAM dapat diakses secara acak oleh CPU.

Penyimpanan

Penyimpanan dalam komputer digunakan untuk menyimpan data secara permanen. Ini termasuk hard disk drive (HDD), solid-state drive (SSD), dan media penyimpanan lainnya.

Input/Output (I/O) Devices

Perangkat Input/Output (I/O) digunakan untuk berinteraksi dengan komputer, seperti keyboard, mouse, monitor, printer, dan perangkat penyimpanan eksternal.
